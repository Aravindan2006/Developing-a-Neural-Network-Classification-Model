# EX 02  Neural Network Classification Model

## 📌 Overview
A Neural Network Classification Model built using PyTorch to classify data into different categories based on input features.

## 🛠️ Technologies
- Python
- PyTorch
- Pandas
- NumPy
- Scikit-learn

## 🚀 Features
- Data preprocessing
- Neural network training
- Classification prediction
- Model evaluation

```bash
pip install -r requirements.txt
python train.py
```
